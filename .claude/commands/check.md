提交前全面檢查：型別、測試、Lint 三關都要通過。

步驟（依序執行，任一失敗立即停止並回報，不繼續往下）：

1. **型別檢查**：執行 `npx tsc --noEmit`
   - 成功：繼續
   - 失敗：列出所有 TS 錯誤（檔案路徑 + 行號 + 錯誤訊息），停止並回報「❌ 型別檢查失敗」

2. **單元測試**：執行 `npm test`
   - 成功：繼續
   - 失敗：列出失敗的測試名稱和錯誤，停止並回報「❌ 測試失敗」

3. **Lint**：執行 `npm run lint`
   - 成功：繼續
   - 失敗：列出 lint 警告/錯誤，停止並回報「❌ Lint 失敗」

全部通過後回報：
```
✅ 全部通過，可以提交
   - 型別檢查：通過
   - 測試：X 個測試全部通過
   - Lint：無錯誤
```

注意：不要跳過任何一關，也不要把警告當成通過。
